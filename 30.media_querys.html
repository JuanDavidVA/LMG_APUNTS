Media Queries

Las Media queries se trata de una característica muy importante de CSS ya que estas nos permiten adaptar el diseño de nuestra página web en función de las características o dimensiones de pantalla de diferentes dispositivos, con el fin de que nuestra web pueda ser visible desde cualquier dispositivos. orientado al desarrollo del responsive par nuestra web.

Que son las media queries

Una media query se trata de una condición que se añade al css para aplicar reglas de estilos que nomas se visualizarán cuando se cumpla la condición indicada 

 
    @media (condición) {
            /*  Aqui adentro van los estilos de la condición*/
 }
Tipos de condiciones más comunes:

Por ancho de pantalla(min-width y max-width): Estas condiciones se utilizan para aplicar estilos cuando la ventana del navegador tiene un ancho mínimo o máximo según lo que se especifique.

 
    @media (min-width: 600px) {
                /* Estilos por pantallas de 600 píxeles o más */
 }
@media (max-width: 600px) {
        /* Estilos por pantalla de hasta a 600 pixeles  */
 }

Por altura de la pantalla (min-height i max-height): Funciona igual que la anterior pero en función de la altura de la pantalla.

 
    @media (min-height: 500px) {
		/*Estilos por pantalla*/
 }

Orientación de la pantalla (orientation): Se puede utilizar para aplicar estilos si la pantalla esta en horizontal o en vertical

 
    @media (orientation: landscape) {
        /*Estilos para pagina horizontal*/
 }
    @media (orientation: portrait) {
        /*Estilos para página vertical*/

 }

Se pueden combinar diversas condiciones en una sola media query mediante las palabras clave and o only para especificar estilos muy concretos 

 
    @media (min-width: 600px) and (max-width: 1200px) {
            /*Estilos aplicados nomas a las pantallas que tiene a 600 y a 1200 de ancho*/
}
