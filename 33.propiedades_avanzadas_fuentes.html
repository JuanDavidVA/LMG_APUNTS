Propiedades avanzadas de fuentes y tipografias

El control de la tipografía de una página web es crucial para garantizar una buena legibilidad y una estética consistente, en el css.

Control avanzado de tipografía

Además de poder definir fuentes de manera personalizada podemos establecer ciertas propiedades para controlar otros aspectos de las fuentes tipográficas

Propiedad letters-spacing

Nos permite ajustar el espacio entre las letras de un texto

 
    h1 {
        letter-spacing: 2px; 
 }
Propiedad line-height

Nos permite controlar el interlineado de un párrafo.

 
    p {
    line-height: 1.6; 
 }
Propiedad font-weight

Nos permite controlar el grosor del texto

 
    p {
        font-weight: 300; 
 }
Propiedad font-style

Nos permite controlar el estilo en que se va a mostrar nuestro fuente,cursiva,normal,oblicua

 
    p {
        font-style: italic;
 }
Propiedad text-transform

Nos permite cambiar la capitalización del texto, convirtiéndolo en mayúsculas,minúsculas o según la primera letra.

 
    p {
        text-transform: uppercase;
 }
Uso del web open Font Format:

Se trata de formatos de fuentes comprimidos diseñados específicamente para el uso web. Este tipo de formato proporciona una compresión más eficiente que los formatos tradicionales.

 
    @font-face {
        font-family: "Roboto";
        src: url("fonts/Roboto-Regular.woff2") format("woff2"),
        url("fonts/Roboto-Regular.woff") format("woff");
 }
