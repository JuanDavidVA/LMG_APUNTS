Popover API

El popover API se trata de una funcionalidad incorporada recientemente que permite a los desarrolladores crear contenido emergente de forma nativa solo con html y CSS. Nos facilita la creación de componentes como menús, o notificaciones que aparecen por encima del contenido existente.

popover atributos
  1. auto: El popover se muestr i se oculta automáticamente
  2. auto: El manual El control de la visibilidad del popover se gestiona de forma manual
popovertargetaction:

Define la acción que se tiene que realizar sobre el popover

  1. show: muestra el popover
  2. hide: oculta el popover
  3. toggle: alternar entre mostrar y ocultar el popover
