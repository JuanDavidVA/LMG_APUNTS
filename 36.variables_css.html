Variables CSS

Las variables en css, también conocidas como Custom properties, permiten definir valores reutilizables dentro de un documento CSS, Esta característica permite facilitar la gestión y mantenimientos de proyectos grandes.

1. Definir y utilizar variables:

Las variables se definen dentro de cualquier selector utilizando el prefijo -- . Es habitual definir los en el selector :root para de que están disponibles globalmente en todo el documento

 
    :root {
    --color-primari: #3498db;
    --marge-petit: 10px;
}
body {
        background-color: var(--color-primari);
        margin: var(--marge-petit);
 }
2.Beneficio de las variables en proyectos grandes

El usos de variables es realmente útil en proyectos grandes y complejos, ya que nos permite gestionar de forma más sencilla el cambio de valores en múltiples lugares manualmente.

Ventajas:
  1. Mantenibilidad
  2. Reutilización
  3. Consistencia
3- Variables CSS con valores por defecto

Se puede proporcionar valores por defecto a las variables CSS en caso que no estén definidas, Esto es esencial a la hora de asegurarse que el diseño funcione incluso si las variables fallan o no se encuentran.

 
    h2 {
            color: var(--color-primari, black);
 }
