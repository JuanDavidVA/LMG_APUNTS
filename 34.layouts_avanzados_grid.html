Layouts avanzados con Flexbox y grid

Flexbox avanzado

Flexbox se trata de una herramienta muy flexible para crear diseños en fila o columna, donde los elementos se alinean y ajustan de forma automática.

Flex-shrink

Esto nos permite controlar cómo se reduce la medida de un elemento cuando no hay casi espacio disponible.

 
    .element {
        flex-shrink: 1; 
 }
Align-self

Esto nos permite que un solo elemento en un contenedor flex este tenga una alineación diferente al resto

 
    .element {
        align-self: center; 
 }
CSS Grid Layout

Se trata de una herramienta más potente que flexbox cuando se trata de crear layouts en dos a más dimensiones, permite la colocación de elementos en cuadrícula con una alta precisión. Propiedades:

Grid-template-columns y grid-template-rows

Nos permiten establecer el número de columnas y filas que deseamos que nuestra cuadrícula tenga

 
    .grid {
        display: grid;
        grid-template-columns: 1fr 2fr 1fr; 
        grid-template-rows: 100px auto;
    }
Grid-column y grid-rows

Nos permite indicarle a un elemento que ocupe diversas columnas o filas en la cuadrícula 

 
    .element-gran {
        grid-column: span 2; 
        grid-row: span 1; 
    }
Gap

Nos permite definir un espacio entre las filas y columnas de la cuadricula

 
    .grid {
        gap: 20px;
 }
Uso combinado de Flexbox y grid

Es bastante útil realizar la combinación de estos 2 para obtener los mejores resultados. Orientando el grid para la estructura general y el flexbox para los elementos interiores de la estructura.
