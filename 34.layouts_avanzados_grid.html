Layouts avanzados con Flexbox y grid

CSS es un lenguaje de estilos el cual esta orientado a realizar el control de la presentación y el diseño de todo un documento HTML como hemos visto en la unidad anterior. CSS nos permite establecer el diseño a nuestro gusto siguiendo la estructura tanto del html como la propia de css.

Ventajas que nos ofrece css:
  1. Diseños mejorados: se puede aplicar estilos a nuestro gusto y establecer una mayor profesionalidad a nuestra web.
  2. Separación del contenido: nos permite separar de los estilos al la estructura html permitiéndonos tener una mejor organización a la hora de trabajar con ellos
  3. Facilidad del mantenimiento: con un único archivo css podemos utilizarlo para aplicar ciertos estilos a diferentes páginas html de tal forma que el mantenimiento sea mucho mas accesibles y fácil de gestionar.
  4. Rendimiento y velocidad: mediante archivos CSS podemos permitir que las web carguen mucho más rápido, de tal forma como comentamos antes con la reutilización de archivos css.
