Propiedades de textos y fuentes

El css nos permite controlar cómo se muestran los textos dentro de un página web. con diferentes propiedades, podemos cambiar la fuente el color,el tamaño…

Propiedad font-family

Esta propiedad define el tipo de letra o tipografía que deseamos utilizar en nuestra elemento de texto indicado

Propiedad font-size

Esta propiedad nos permite controlar el tamaño de letra el cual desearemos utilizar para nuestro web existen diversas unidades par su utilización como (px),,(ems), (pt),(%).

Propiedad color

Esta propiedad nos permite definir el color de texto el. cual deseamos utilizar los colores se pueden establecer con rgb o rgba

Propiedad text-align

Esta propiedad nos permite controlar el tipo de alineación de texto que queremos utilizar para el elemento, existen diferentes tipos como left, right, center, justify.

Propiedad font-weight

Esta propiedad nos permite controlar el ancho de la letra

Propiedad font-style

Esta propiedad nos permite controlar el estilo de la fuente
