Selector avanzado

El selector avanzado en CSS nos permite aplicar estilos con mucha más precisión y flexibilidad, permitiendo seleccionar elementos específicos en función de su posición.

Selector de atributos

Este nos permite aplicar estilos a los elementos HTML que contienen un determinado atributo o valor de atributo,Son especialmente útiles para dar estilos a formularios

Seleccionar elementos con un atributo específico:
 
    input[type="text"] {
    border: 1px solid blue;
}
Selectores combinados

Los selectores combinados permite la selección de los elementos que tiene una relación específica entre elementos dentro de un documento, combinaciones comunes:

Combinador descendente
 
    div p {
    color: red;
 }
Combinador de hijos directos
 
    div > p {
        color: blue;
 }
Combinador de hermanos adyacentes
 
    h1 + p {
            font-weight: bold;
 }
Combinador de hermanos generales 
 
    h2 ~ p {
        color: gray;
 }
Selectores de pseudoclases

Las pseudo-clases son selectores que se utilizan para aplicar estilos a un elemento en un estado o condición particular:

:hover Aplica estilos cuando el usuario pasa el ratón sobre el elemento

 
    a:hover {
    color: red;
 }

:nth-child() Selecciona elementos en función de su posición entre sus hermanos

 
    tr:nth-child(odd) {
        background-color: #f0f0f0;
}

:first-of-type Selecciona el primer elemento de su tipo dentro de su padre

 
    p:first-of-type {
        font-weight: bold;
}

:not() Selecciona todos los elementos menos aquellos que coinciden con el selector especificado.

 
    div:not(.important) {
        background-color: yellow;
}
Selector pseudo-elemento

:not() Estos permiten aplicar estilos a una parte especifica de un elemento,com su contenido antes o despues, o una línea de texto:

pseudo-elemento :before

Añade contenido o estilos antes del contenido real de un elemento

 
    h1::before {
    content: "★ ";
color: gold;
}
pseudo-elemento :after

Añade contenido o estilos después del contenido real de un elemento

 
    p::after {
    content: " (Final)";
color: gray;
}
