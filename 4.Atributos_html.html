Atributos html:

Los atributos se tratan de pequeños elementos que nos permiten añadir información extra a las
distintas etiquetas de html para realizar distintas funciones. Forma de utilización es establecer
el nombre del atributo y el valor dentro de la etiqueta inicial de cada elemento html
(nombreAtributo="valor")

Ejemplo:

  • Anterior apunte
  • Siguiente apunte

Acerca de:

  • Sobre mi(Curriculum)
  • Objetivos de la web
  • Institución pertneciente
