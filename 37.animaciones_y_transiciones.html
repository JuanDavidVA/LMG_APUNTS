Animaciones y transiciones

Las animaciones y transiciones en CSS son herramientas muy potentes que permiten añadir movimiento e interacción visual a las páginas web con el fin de mejorar la experiencia de usuario.

Transiciones

Las transiciones permiten que las propiedades de un elemento se visualice com mayor suavidad:

 
    button {
background-color: lightblue;
transition: background-color 0.3s ease;
}
2.Propiedades de transición

Se pueden especificar diversas propiedades para animarlas simultáneamente con transition-property o utilizar all para aplicar a la transición todas las propiedade. 

 
    div {
width: 100px;
height: 100px;
background-color: lightgreen;
transition: width 1s ease, background-color 0.5s ease-in-out;
}
3.Animaciones clave con @keyframe

Nos permiten crear movimientos mas complejos que las transiciones. Utilizando la regla @keyframe para definir puntos intermedios del movimiento o animación.

 

Sintaxis de keyframe

    @keyframes nombre-animación {
0% { propietat: valor-inicial; }
100% { propietat: valor-final; }
}


 

Sintaxis de animaciones

    element {
animation: nombre-animación duración función-temporización retraso ciclos
dirección;
}
Propiedades de animaciones

Estas nos permite personalizar cómo se comportan las animaciones:

  1. animation-duration:controla el tiempo de duración
  2. animation-delay: define un pequeño retraso antes que la animación comience
  3. animation-iteration-count: Controla cuántas veces se repite la animación
  4. animation-direction: define si la animación va hacia delante o atrás o alterna
