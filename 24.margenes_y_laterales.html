Margenes y laterales

Los márgenes, paddings y borders son un parte de el modelo de las cajas en CSS, que describen la distribución de elementos en una página

1- Márgenes

Nos permite controlar el espacio exterior de un elemento, el espacio entre el elemento y otros elementos.

 
 p {
          margin: 10px; /* Para aplicar margenes de 10 pixeles a cada lado */
  }
2- Padding

Esta propiedad padding nos permite controlar el espacio interior de un elemento, el espacio entre el contenido de un elemento y sus propios bordes., .

 
  div {
        padding: 25px; /*Aplica un padding de 25 px a cada lado */
    }
3- bordes

Se utiliza para añadir un borde alrededor de un elemento

 
    div {
        border: 2px solid black; /* borde negro de 1px solido */
      }
      p {
         border: 1px dashed blue; /* Le indicamos un borde de 1 px rallado azul */
      }

Existen diversas unidades disponibles para utilizar:

  1. Pixels(px)
  2. Ems(em)
  3. Rems(rem)
  4. Porcentajes(%)
  5. Viewports(vw,vh)
  6. Bordes(border-width)
