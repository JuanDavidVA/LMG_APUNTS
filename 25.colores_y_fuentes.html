Colores y fuentes

En css nos proporciona diversas propiedades para aplicar colores y fuentes a los elemento HTML

La propiedad color se utiliza para cambiar el color del texto dentro de un elemento, se puede definir de diversas maneras:

  1. Colores por nombre, ejemplo: green, red, blue
  2. Colores por hexadecimal, #FF0000
  3. Colores por nombre, ejemplo: green, red, blue
  4. Colores por RGB, rgb(255,255,255)
  5. Colores por RGBA, rgb(255,255,255, 0)
 
    p {
        color: blue; /* Utilizando colores por nombre */
    }

    h1 {
         color: #FF5733; /* utilizando colores por hexadecimal */
    }

    span {
            color: rgb(0, 255, 0); /* Colores por RGB*/
    }

    div {
         color: rgba(0, 0, 255, 0.5); /* colores por RGBA */
     }
La Propiedad background-color

Se utiliza para establecer el color de la fuente de un elemento

 
    body {
         background-color: lightgray; /* Color del fondo gris */
    }
La propiedad background

Se trata de una manera abreviada de especificar diversas propiedades de la fuente 

 
     div {
        background: lightblue url("imagenEjemplo.jpg") no-repeat center cover;
                 /*nos permite establecer el fondo*/
    }
La background-image

Nos permite añadir una imagen como fuente

 
     div {
        background-image: url("imagenEjemplo.jpg");
    }
  1. Background-repeat: Controla si la imagen de fuente se repite
  2. Background-position: Define la posicion de la imagen de fuente
  3. Background-size: Permite ajustar el tamaño de imagen
 
     div {
        background-image: url("imagenEjemplo.jpg");
        background-repeat: no-repeat;
        background-position: center;
        background-size: cover;
     }
