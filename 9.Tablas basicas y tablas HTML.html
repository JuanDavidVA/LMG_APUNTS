Etiquetas para tablas basicas

  • Etiqueta (<table>)

    Esta etiqueta nos permite establecer el contenedor principal de la tabla.

  • Etiqueta (<tr>)

    Esta etiqueta nos permite establecer una fila de nuestra tabla.

  • Etiqueta (<td>)

    Nos crea una celda en la fila creada en la cual podemos introducir datos.

Atributos basicos de una tabla

  1. Atributo border:

    Este atributo nos permite establecer un borde a nuestra tabla

  2. Atributo cellspacing

    Este at atributo nos permite controlar el espacio entre celdas

  3. Atributo cellpadding

    Este atributo nos permite controlar el espacio entre contenido y el lateral de la celda

Tablas con html

  • Etiqueta (<thead>)

    Esta etiqueta nos permite establecer una cabezera para toda una fila (visual).

  • Etiqueta (<tbody>)

    Esta etiqueta nos permite establecer el cuerpo de nuestra tabla, permite separar los datos de la cabezera.

  • Etiqueta (<th>)

    Nos permite crear una celda en el encabezado <thead> de la tabala.
