Posicionamiento CSS

Propiedad position

La propiedad position nos permite especificar el método de posicionamiento de un elemento, según el valor que elegimos:

La propiedad top, right, bottom, left, cuando establecemos un posición podemos utilizar dichas propiedades de posicionamiento para especificar la distancia del elemento respecto sus límites.

Propiedad relative

Se posiciona respecto a su posición original en el flujo del documento. Se puede utilizar para mover por coordenadas un elemento. 

 
    div {
            position: relative;
            top: 20px; 
            left: 30px; 
 }
Propiedad position absolute

Con esta propiedad, el elemento sale completamente de su flujo normal del documento y se posiciona respecto a su elemento contenedor más próximo a un position relative o fixed.

 
    div {
            position: absolute;
            top: 50px;
            left: 100px;
 }
Propiedad position fixed

Dichos elementos se mantienen en una ubicación fija en la pantalla incluyendo cuando se realizan desplazamientos por la pantalla. 

 
    div {
            position: fixed;
            top: 0;
            left: 0;
            width: 100px;
            height: 100px;
 }
Propiedad position sticky

Se comportan como relative hasta que el usuario desplaza la pantalla luego el elemento se transforma en un fixed.

 
    h1 {
        position: sticky;
        top: 21px;
 }
Propiedad z-index

Nos permite controlar el orden de superposición o la capa en que se encuentra un elemento 

 
    div {
        position: absolute;
        z-index: 10; /*Este elemento se muestra delante de los otros */
 }
