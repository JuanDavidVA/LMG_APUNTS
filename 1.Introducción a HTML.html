Introducción html:

HTML se trata de un lenguaje de marcas orientada a la creación y estructuración de páginas web,
a partir de un conjunto de etiquetas las cuales son interpretadas por el navegador estas componen
a la estructura básica de la página web.

(HyperText Markup Language)

El lenguaje de marcas HTML fue creado por Tim Bernerss-Lee el 1991, HTML 2.0 fue el primer
estándar oficial en el año 1995, posteriormente se estableció la última versión HTML5 incluyendo
nuevas funcionalidades como soporte para video, audio y gráficos.
