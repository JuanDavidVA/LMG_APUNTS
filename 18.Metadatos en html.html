Metadatos:

Los metadatos se trata de información que se establece dentro de nuestro documentos html con el objetivo de dar información importante al documento. Todos los metadatos se deben de establecer en el head de nuestra página web

  1. Charset

    Permite definir qué conjunto de caracteres se utilizaran en nuestr web

  2. Viewport

    Se trata de un metadato el cual nos ayuda a la visualización de nuestra página web en dispositivos con diferentes tamaños de pantalla(responsive)

  3. Descripción

    Nos permite realizar un breve descripción del contenido de la pagina

  4. Keywords

    Nos permite indicar una serie de palabras claves para nuestra pagian web que benefician al SEO

  5. Author

    Nos permite indicar en nombre del autor o creador o encargado de dicha página html
