Sintaxis basica

  1. Una regla: css se trata de la instrucción que define cómo se debe aplicar un estilo a uno o varios elementos.
  2. Selector: nos permite indicar el elemento al cual deseamos aplicarle estilos.
  3. Propiedad: se define la característica que deseamos modificar
  4. valor: se le asigna el valor específico que deseamos establecer
