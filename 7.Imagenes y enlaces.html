Imagenes en HTML

Para implementar una imagen en nuestra pagian web utilizamos <img > esta etiqueta tiene la particularidad de que no contiene etiqueta de cierre, para poder establecer imágenes debemos utilizar dicha etiqueta más los siguientes atributos.

src(source):

Este nos permite establecer la ruta tanto relativa o completa de nuestra imagen.

alt(alternate text):

En caso de que nuestras imágenes no se puedan cargar en la página web aparecerá un texto alternativo el cual nosotros definiremos.

Width i height:

El width nos permite establecer la anchura de nuestra imagen y height la altura.

Enlaces en HTML:

Para realizar la implementación de enlaces tendremos la etiqueta <a> esta la podrás utilizar tanto en enlaces externos como enlaces dentro de nuestro mismo proyecto o secciones de nuestras páginas. El contenido que establezcamos entre nuestras etiquetas de apertura y cierre será el texto que se visualizará para dicho link en nuestra página web. Para poder implementar nuestros enlaces debemos establecer ciertos atributos para su funcionamiento:

href (Hypertext reference):

Nos permite establecer la ruta URL o establecer una ruta relativa

target:

El atributo target tiene diferente valores donde los mas imprescindibles se tratan de _blank (Nos permite abrir en enlace en otra pagina) y _self (nos abre el enlace en la misma página)f
