Etiqueta (<template>)

La etiqueta template nos permite definir fragmente s html reutilizables evitando grandes repeticiones de contenido, pero este template no se inserta dentro de la página hasta que sea modificado con javascript, beneficiosos para las páginas dinámicas

Se debe de tener en cuenta a realizar siempre las pantallas lo más simples posibles de forma que permita la reutilización fácilmente, tener en cuenta la necesidad de javascript.
