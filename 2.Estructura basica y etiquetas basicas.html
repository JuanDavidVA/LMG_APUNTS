Estructura basica

Html se basa en una estructura jerárquica, el cual está compuesta por una serie
de etiquetas básicas que componen a dicha estructura, a partir de esta se pueden
obtener estructuras más complejas:

En la primera línea tenemos el <!DOCTYPE html> se trata de una etiqueta única el
cual nos permite indicar la versión actual de html.

Posteriormente tendríamos una de las etiquetas más importantes el cual se encarga
de contener todas las etiquetas de html es <html></html>.

En cuanto a el <head></head> este se encarga de contener todas las etiquetas
orientadas a los metadatos, contenido importante el cual no se visualiza por
pantalla de la web pero importante para el funcionamiento correcto de la pàgina web.

Por último tendríamos el body el cual es el encargado de contener toda la estructura
visual de nuestra página web todas las etiquetas que se establezcan aqui se visualizarán.

Etiquetas basicas

En primer lugar tendríamos <title> dicha etiqueta nos permite establecer el
nombre que aparece en la pestaña superior de la web

Otra de las etiquetas básicas se trataría de los <h1$gt; a <h6> estas nos
permite establecer títulos y subtítulos de una web

La etiqueta <p> nos permite establecer un párrafo dentro de la web

La siguiente etiqueta <br> nos permite realizar un salto de línea

Como última etiqueta básica & nbsp; dicha etiqueta nos permite insertar un espacio
en blanco entre líneas. Nos permite mantener la unión entre 2 o masa palabras de
forma, esto nos permite mejorar a la hora de utilizar el responsive
